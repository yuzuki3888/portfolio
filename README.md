# ポートフォリオテンプレート

情報デザイン学部　ポートフォリオⅠ〜Ⅳ で使うテンプレートです。

---

## このリポジトリの使い方

1. 右上の「Use this template」ボタンをクリックする
2. 自分のアカウントに`portfolio`という名前でリポジトリを作成する
3. 以下のファイルを使ってふりかえりを記録していく

---

## ファイルの説明

- `subject-review.md` → 科目ごとのふりかえりに使います。科目の数だけコピーして使ってください
- `term-review.md` → 学期末の総合ふりかえりに使います。学期ごとに1つ作成してください

---

## わからないときは

担当教員またはTAに連絡してください。
